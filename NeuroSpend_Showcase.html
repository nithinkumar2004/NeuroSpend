Clean Architecture

Every layer is intentional, testable, and scalable from day one

📱
SwiftUI Views
7 screens with glassmorphism cards, Swift Charts, smooth animations, and dark mode. Zero UIKit dependency.
SwiftUI Swift Charts Combine
🏗
MVVM ViewModels
5 @MainActor ViewModels with @Published state. AuthVM, ExpenseVM, AIInsightsVM, BudgetVM, SubscriptionVM.
@MainActor async/await @Published
🔥
Firebase Backend
Firestore with real-time listeners, Firebase Auth (Email + Apple), and Cloud Functions as secure AI proxy layer.
Firestore Firebase Auth Cloud Functions

Claude AI Layer
Expense categorization, spending analysis, monthly predictions, fraud detection, and smart financial advice.

Security First

AI Prompt Engineering

Production-grade prompts designed for accuracy and structured JSON outputs

→ Expense Categorization
You are a financial categorization AI. Categorize the following expense into EXACTLY one of these categories: Food & Drinks, Transport, Bills, Entertainment, Health, Shopping, Education, Travel, Subscription, Other. Expense details: - Description: Starbucks coffee - Amount: 6.50 Rules: - Return ONLY the category name, nothing else - Match spelling exactly as listed above - Use context: "Netflix" → Subscription, "Uber" → Transport
→ Spending Analysis (3 Insights)
You are a personal finance AI analyst. Analyze this spending data and provide exactly 3 financial insights. Spending Data: Total: 1247.80 | Food: 312.00 (25%) | Shopping: 220.00 (18%) | Transport: 148.00 (12%) Monthly Budget: USD 1500.00 Return JSON array: [{"title":"...","description":"...","type":"spending_pattern|prediction|advice|anomaly","severity":"info|warning|critical"}] Each description under 50 words. Be specific with percentages.
→ Fraud / Anomaly Detection
You are a financial fraud detection AI. Analyze these transactions for anomalies. Transactions: ID:abc123 Amount:340.00 Category:Shopping Date:2025-12-29T14:23:00Z ID:def456 Amount:6.50 Category:Food Date:2025-12-29T08:11:00Z ... Flag transactions: >3x category average, duplicates within 24h, unusual patterns. Return ONLY: {"flaggedIds": ["abc123"]}
